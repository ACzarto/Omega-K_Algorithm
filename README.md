# Omega-K_Algorithm
This repository contains a type of Synthetic Aperture Radar (SAR) Image Processing Algorithm, the so-called Omega-K algorithm that operates the image formation in the frequency domain using the Stolt interpolation.

In this code we generate the synthetic raw data (supposing 3 corner reflectors on the soil) and then apply the algorithm to test if the image was correctly generated.
